# PYNN

## About

**pynn** - Neural Network Library

## Install

```shell
$ pip install pynn
```

## Getting Started

```python
import pynn

# New returns a new neural network
# instance with the default parameters.
n = pypynn.new()

# Dataset.
data_data_input  = {.27, .31}
data_data_target = {.7}

# Training dataset.
_, _ = pypynn.train(data_data_input, data_data_target)
```
